## Create standalone editor application

You can create a standalone editor application which is connected to a LLM. The application supported:

* Rich text editor
* AI integrated. So, you can use it to summarize, generate text, fix grammar, etc.
* Persist the data.
* Export to PDF, Markdown, etc

[![Import Project](./docs/editor-application.gif)](./docs/editor-application.gif)
**Example:**: Editor Application help fix the grammar of the text.
