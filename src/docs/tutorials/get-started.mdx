## Get Started Flow

You can do as follows to get started with WebLLM:

Create a LLM node by using toolbar.

[![LLM Node](./docs/create-web-llm.gif)](./docs/create-web-llm.gif)

Create a Prompt node by using toolbar.

[![Prompt Node](./docs/create-a-prompt.gif)](./docs/create-a-prompt.gif)

Load LLM model and create a new thread.

[![LoadLLM](./docs/load-llm-and-create-thread.gif)](./docs/load-llm-and-create-thread.gif)

Connect Prompt node to Thread node.

[![Connect Prompt to Thread](./docs/prompt-to-thread.gif)](./docs/prompt-to-thread.gif)
