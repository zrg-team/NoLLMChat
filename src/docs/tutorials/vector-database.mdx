## Vector Database as a retrieval

With this example you can see how to use a vector database as a retrieval system. LLM can use the vector database to retrieve the most similar context and anwsers to the given prompt.

Currently the vector database support following:
* Indexing of text or pdf or text files.
* Persisting the indexed data.
* Context splitting.
* Querying the indexed data.
